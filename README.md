## Help
```bash
dcli --help
```

## Install from sources
```bash
git clone https://github.com/deadlock-resources/dcli.git
cd dcli/
pip3 install .
```

## Generate mission
```bash
dcli gen (java|python)
```
## Execute mission
```bash
dcli solve ./mission_path
dcli run ./mission_path
```
